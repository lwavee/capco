## 🚀 Key Features

| Feature | Description |
| :--- | :--- |
| **Modular Architecture** | Reusable Header/Footer components injected via JavaScript for maintainability. |
| **Dynamic Pathing** | Intelligent path resolution logic allowing the same components to work in root and subdirectories. |
| **SEO Mastery** | Full implementation of Meta tags, Open Graph protocols, and JSON-LD Schema markup. |
| **Premium Aesthetics** | Modern typography, curated color palettes, and subtle micro-animations using Intersection Observer. |
| **Responsive Design** | Fully fluid layouts optimized for mobile, tablet, and ultra-wide desktop displays. |
| **Performance** | Clean, vanilla code without heavy dependencies, ensuring lightning-fast load times. |

## 📁 Project Structure

```text
capco/
├── index.html            # Main Landing Page (Home)
├── style.css             # Page-specific styles
├── script.js             # Page-specific logic
├── shared/               # Global assets and styles
│   ├── global.css        # Design System & Design Tokens
│   └── global.js         # Core animations & utility scripts
├── components/           # Reusable UI Modules
│   ├── header/           # Dynamic Navigation System
│   └── footer/           # Global Footer Module
└── [Service Pages]/      # Specialized industry pages (e.g., General Liability)
```
